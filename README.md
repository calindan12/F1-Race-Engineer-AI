# F1 Race Engineer AI

An AI-powered Formula 1 analysis platform that uses multiple specialized agents to collect race data, analyze driver performance, evaluate race strategies, and generate professional race reports.

## Features

- LangChain Tool Calling
- Multi-Agent Architecture
- LangGraph Workflows
- CrewAI Collaboration
- LangSmith Observability
- External F1 APIs
- Tool Registry (MCP-inspired)
- Automated Race Reports

## Workflow

User Query
    ↓
Data Agent
    ↓
F1 API Tools
    ↓
Strategy Agent
    ↓
Race Engineer Agent
    ↓
Final Race Report
